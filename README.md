### 前期调试遇到的一些问题
* 车剧烈抖动
* 蓝牙收发卡死
* 串口堵死
### 解决办法
* 在调试直立环的时候车剧烈抖动，坚持不过2s，起初我以为是PID的问题，加上又写了蓝牙调试器这样方便的东西，就调了一天的PID.
